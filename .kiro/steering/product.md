# Product Overview

Modern portfolio website for Melnar Ancit Cordova, a CRM & Software Engineer.

## Purpose

Showcase professional work, technical skills, projects, and blog content with a focus on clean design and optimal performance.

## Key Features

- Dark theme with neon green accents
- Fully responsive design (mobile, tablet, desktop)
- Static site generation for performance
- Protected resume download with authentication
- Contact form with email integration (Resend API)
- Blog with markdown support and syntax highlighting
- Accessibility-focused components

## Target Audience

Potential employers, clients, and collaborators looking to learn about the developer's skills and experience.
